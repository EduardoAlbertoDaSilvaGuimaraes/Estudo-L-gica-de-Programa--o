Começo do estudo de lógica de programação

Porque um software é criado?

Ele cria soluções para os problemas do dia a dia.

Criado para automatizar e otimizar processos.
Pretende automatizar processos repetitivos, exemplos: automatizar tarefas na web(interagir,preencher ou buscar informações em sites)

Como um software é criado do zero e onde a lógica encaixa nisso?

1 Ocorre uma ideia para desenvolver um software

2 logo como segundo passo vem a especificação do software para saber oque ele deve ou não deve resolver
3 A validação da solução é mostrar para o cliente a idealização do software na sua visão para ter a validação para seguir com o projeto
4 Começo de desenovlvimento de codigos
5 O software está funcional e pronto para a entrega, é entregue para o cliente e pode ser feita novas implantações no projeto
